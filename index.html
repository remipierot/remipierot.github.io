Formulas
- x = cos(a * t * 2PI) - cos(b * t * 2PI)^j
- y = sin(c * t * 2PI) - sin(d * t * 2PI)^k
Details
- a, b, c, d, j, k can be arbitrarily chosen
- If b = 0, j will not have any effect
- If d = 0, k will not have any effect
- r is the number of points used to draw the curve
- Randomize will not change r
